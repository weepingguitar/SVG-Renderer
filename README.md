## Installation

1. Run command:
  ```shell
      git clone https://github.com/weepingguitar/SVG-Renderer
      cd SVG-Renderer
  ```
## Compile

1. Xmake is needed. Run:
  ```shell
    xmake
    xmake run svg
  ```
